Detailed Description

  • NAME Game.cs
  • DESCRIPTION This class holds the data structure responsible for keeping track of the state of the board. It handles user input, turn order, validates the player move by determining the appropriate validation method from Rule.cs to call It updates the board by looping through the board and passing the appropriate parameters for each square to GameDisplay::SetButtonImage
  • AUTHOR Daniel Melnikov
  • DATE 9/12/15
